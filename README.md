# homebrew-mermaid

Homebrew tap for [Mermaid](https://github.com/noahsabaj/mermaid-cli) — an open-source, model-agnostic AI pair programmer for the terminal.

```bash
brew install noahsabaj/mermaid/mermaid
```

That's shorthand for `brew tap noahsabaj/mermaid && brew install mermaid`. Upgrade with `brew upgrade mermaid`.

The formula is bumped automatically on each [mermaid-cli release](https://github.com/noahsabaj/mermaid-cli/releases); please don't edit it by hand.
